This blog is hosted at GitHub Pages and constructed using Publii.

GitHub Pages is dead-simple and free for personal blogs.

This blog is a static website. All pages are served straight from GitHub Pages, with no other server-side processing.

I started an old blog a few years ago using Jekyll, but I grew tired of wondering whether Jekyll would be maintained very far into the future. I've finally switched to Publii instead; it seems a little more modern. We'll see!

When I publish a post, I intend not to update it. If an existing post needs updates, I'll mark up the post to show what has been changed.
